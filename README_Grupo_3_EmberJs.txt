Descripción: 

	Se adjunta las carpetas de los proyectos y el backup de la base de datos.
	
	BACKEND = Demo005_Mangas_ApiRest
	FRONTEND = mangas
	BACKUP = manga.bak

Requisitos:

	- Se necesita tener instalado NodeJs

	- Instalar el CLI de Ember con el comando
	
		npm install -g ember-cli

	- Para levantar el proyecto de ember hay que ejecutar el siguiente comando
	 dentro de la carpeta del frontend.

		ember serve
